# obiameixendacapeans-cyber.github.io
En esta tarea he creado una página web y la he alojado en GitHub Pages utilizando HTML, CSS y JavaScript. El repositorio está organizado con las carpetas images, styles y scripts, además de los archivos principales como index.html, README.md y LICENSE. En la web he añadido varios elementos interactivos con JavaScript, como el intercambio de imágenes al hacer clic sobre ellas, un campo de texto que cambia el contenido de la página según lo que escriba el usuario y un objeto en movimiento. También he incorporado una fuente llamativa mediante Google Fonts, enlaces externos que se abren en una nueva pestaña y una estética actual inspirada en páginas informativas sobre el cambio climático. Por último, la web está correctamente publicada y accesible desde el enlace de GitHub Pages incluido en este repositorio.
